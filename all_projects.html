WiFi LED Controller

Completion: 2025

An LED controller that can be accessed via its own WiFi access point or connected to a home network. The web UI, provided by the ESP32 microcontroller, allows to set static colors or to apply various patterns, like a rainbow mode. The UI focuses on simplicity and usability to allow children to control the LEDs as well. I assembled some of these LED controllers for family and friends, featuring a 1.5m LED strip. However, various lengths should be supported.

Technologies Used

  • ESP32 microcontroller
  • Rust (ESP-IDF)
  • WS2812 NeoPixel LED Strip
  • HTML, CSS, JavaScript

Related Topics

  • IoT Systems
  • WiFi (AP, STA)
  • Embedded Web Servers
  • Low barrier User Experience

Adaptive Signal Filtering

Completion: 2020 (Bachelor Thesis)

Research, development, and implementation of adaptive filtering algorithms for real-time noise reduction in edge computing. This project formed the core of my bachelor's thesis at the Technical University of Vienna. The system is designed for low-latency signal processing, particularly for audio applications, using a heterogeneous hardware architecture: an MCU for real-time signal analysis, a server for dynamic filter configuration, and an FPGA for high-performance filtering. The devices communicated via a custom UART-based protocol. Key findings were published in IEEE Xplore.

Technologies Used

  • Altera FPGA
  • VHDL for filter implementation
  • Teensy microcontroller
  • Arduino flavored C for spectral analysis
  • MATLAB for computing filter coefficients

Related Topics

  • Digital signal processing
  • Communication systems
  • Noise reduction techniques
